# KV Cache Compression Trade-off Analysis

This repository contains code and experimental results for analyzing KV cache compression methods.

## Methods
- Baseline
- KVZip
- Top-K

## Datasets
- KorQuAD
- NarrativeQA

## Results
KVZip achieves higher accuracy but increases latency and memory usage, highlighting a trade-off between efficiency and performance.

## Repository Structure
- src/: evaluation scripts
- results/: experiment outputs
- *.png: figures used in the paper
